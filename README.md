# SO
Trabalhos da Disciplina de Sistemas Operacionais

### `make` or `make all`

* Compila o programa a partir de `src`, gera objetos em `obj` e cria um executável em `bin`

### `make run`

* Executa o programa `bin/alba`
* 
### `make clean`

* Apaga arquivos desnecessários
